# Project 1: Cloud Risk & Compliance
## ISO 27001 + SOC 2 + NIST CSF

### Scenario
NovaPay is a fictional cloud-based payment SaaS startup preparing for
ISO 27001 certification and SOC 2 Type 1 readiness assessment.
As GRC Analyst, I conducted a full risk assessment and compliance gap analysis.

### Artifacts
| File | Description |
|------|-------------|
| asset-inventory.xlsx | 28 assets classified by type, owner, and CIA ratings |
| risk-register.xlsx | 28 risks with likelihood, impact, treatment, and ISO 27001 control mapping |
| statement-of-applicability.xlsx | All 93 ISO 27001:2022 Annex A controls with applicability and status |
| gap-assessment-report.pdf | Executive gap report with remediation roadmap |
| risk-dashboard.xlsx | Visual risk summary with charts |

### Frameworks Used
- ISO/IEC 27001:2022
- SOC 2 Trust Services Criteria
- NIST Cybersecurity Framework (CSF)

### Key Findings
- 6 Critical risks identified in Payment Processing and Cloud Infrastructure domains
- 34 ISO 27001 controls partially implemented; 12 not yet started
- Estimated 6-month roadmap to certification readiness
