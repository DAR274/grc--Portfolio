# Project 3: AI-Powered GRC Automation

### Scenario
Demonstrating how AI tools (Claude, ChatGPT) can accelerate GRC workflows
while maintaining human oversight and quality control.

### Artifacts
| File | Description |
|------|-------------|
| ai-prompt-library.md | 20 tested GRC prompts with outputs |
| policy-comparison.pdf | AI draft vs human-reviewed policy (2 policies) |
| audit-checklist.xlsx | AI-assisted ISO 27001 internal audit checklist |
| risk-register-generator.xlsx | Semi-automated risk register with dropdown inputs |

### Key Insight
AI reduced policy drafting time by ~70% and risk identification time by ~50%.
Human review remained essential for context, accuracy, and regulatory nuance.
