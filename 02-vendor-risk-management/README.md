# Project 2: Third-Party Vendor Risk Management (TPRM)

### Scenario
NovaPay onboards 10 third-party vendors. I designed and implemented
a TPRM program to classify, assess, and monitor vendor risk.

### Artifacts
| File | Description |
|------|-------------|
| vendor-inventory-tiering.xlsx | 10 vendors classified into 3 tiers |
| vendor-risk-questionnaire.pdf | 35-question security assessment questionnaire |
| vendor-risk-scorecard.xlsx | Risk scores and ratings for all vendors |
| tprm-policy.pdf | Formal TPRM policy document |
| vendor-dashboard.xlsx | Vendor risk heatmap and monitoring tracker |

### Key Findings
- 2 Tier 1 vendors identified with critical data access
- 1 vendor flagged High Risk due to no ISO 27001 certification and poor incident response SLA
- Recommended quarterly reviews for Tier 1, annual for Tier 3
